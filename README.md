# Projeto Anna Bella

Site fictício de uma modelo desenvolvido para praticar HTML e CSS.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 📌 Sobre o projeto
Este projeto foi criado com o objetivo de treinar conceitos básicos de desenvolvimento web, como estruturação com HTML e estilização com CSS.

## 🎯 Funcionalidades
- Página inicial
- Biografia
- Campanhas publicitárias
- Contato
